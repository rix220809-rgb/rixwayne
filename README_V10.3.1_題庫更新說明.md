## 每日必答是否會自動變成未來題目？
會。V10.3.1 已真正接上自動轉換流程：

1. 蕭小舜與懷寶必須都回答同一天、同一題每日必答。
2. 完成後等待至少 3 天。
3. 網站初始化時會讀取 Supabase `daily_couple_answers`。
4. 系統會把雙方答案組成一題「每日必答回顧」開放題。
5. 這些題目會自動加入一般混合遊戲題庫。
6. 不會寫入 `events.json`，也不會生成首頁回顧卡。
7. 最多載入最近 60 組完整雙方回答。

上一版只有寫好 `generatedQuestionsFromCoupleAnswers()`，但沒有呼叫它，
所以實際上不會自動加入題庫；本版已補上這個缺口。
