# Our Memories V10.3 — Unified Cycle System

## 避孕藥規則
- 經期 Day 1～4：暫停提醒。
- Day 5：避孕藥第 1/28 天。
- Day 6：第 2/28 天。
- Day 32：第 28/28 天。
- 即使經期已結束，提醒仍持續到第 28 天完成。
- 下一次經期開始後，舊週期立即重設；Day 1～4 再次暫停，Day 5 重新開始。

## 安裝
1. 完整覆蓋 GitHub repository。
2. 用本包 `supabase/functions/send-daily-reminders/index.ts` 覆蓋 Edge Function。
3. 按 Deploy updates。
4. 等 GitHub Pages 部署完成。
5. 手機若仍顯示舊版，刪除主畫面 App 與 Safari 網站資料後重新加入。
